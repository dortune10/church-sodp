# RCCG SODP Church Website

A comprehensive church website and management system built with Next.js, Supabase, and Tailwind CSS.

## Features

- **Public Website**: Information about ministries, sermons, events, blog, and gallery.
- **Admin Dashboard**: Secure management of church members, content, and media.
- **Authentication**: Role-based access control (Admin, Editor, Leader).
- **Gallery**: Photo albums with local storage integration.
- **Sermons**: Media library for audio and video sermons.
- **Events**: Calendar and event registration system.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- **Frontend**: Next.js 15 (App Router), React 19, Tailwind CSS
- **Backend**: Supabase (Auth, Database, RLS)
- **Database**: PostgreSQL
- **Media**: Local storage for gallery images
